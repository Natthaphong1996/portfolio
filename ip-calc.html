Common Cisco Commands

Essential commands for managing Cisco Network devices

🔍 Verification

Check Interface Status
show ip interface brief 📋
Summary of IP and status for all interfaces
View Running Config
show running-config 📋
Display current active configuration in RAM

🛠️ Configuration

Enter Config Mode
configure terminal 📋
Switch to Global Configuration mode
Assign IP to Interface
ip address [IP] [MASK] 📋
Set IP and Mask for a specific port

💾 Maintenance

Save Configuration
write memory 📋
Save changes to non-volatile memory
Restart Device
reload 📋
Reboot the switch or router

🏷️ VLAN Management

Create VLAN
vlan [ID] 📋
Create a VLAN ID and assign a name
Assign Port to VLAN
sw access vlan [ID] 📋
Set interface to Access mode and assign to VLAN
Verify VLANs
show vlan brief 📋
Display brief summary of all configured VLANs

🛤️ Routing Configuration

Static Routing
ip route [Net] [Mask] [Next-Hop] 📋
Manually configure a path to a destination
Default Route
ip route 0.0.0.0 0.0.0.0 [NH] 📋
Set the gateway of last resort
Check Routing Table
show ip route 📋
Display the current IP routing table

OSI 7-Layer Model

Standard framework for network communication and protocols

Layer 7

Application

Interfaces with users and software (HTTP, FTP, SMTP)

Layer 6

Presentation

Data processing, encryption, and compression

Layer 5

Session

Managing and controlling connections between apps

Layer 4

Transport

End-to-end data delivery and error control (TCP, UDP)

Layer 3

Network

Routing and logical addressing (IP Address)

Layer 2

Data Link

Physical addressing and LAN switching (MAC Address)

Layer 1

Physical

Cables, connectors, and electrical signals
